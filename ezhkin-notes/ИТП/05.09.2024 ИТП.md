**Типы хранения данных:**

 - Прямой (используется только для положительных чисел):

	 75 = 01001011
	 -75 = 11001011
	0 = 00000000
	-0 = 10000000

 - Обратный:

	75 = 01001011
	-75 = 10110100

	100 = 01100100
	-25 = 11100110
	При сложении: 101001010
	Перемещаем 1 в начало и получаем 01001011 = 75

 - Дополнительный код (добавление 1 не трогая старший разряд):

	-75 = 10110100
	    +01001011 (75)
			  +1
	  = 10000000 = 0
